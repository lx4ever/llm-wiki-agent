# Skims风黑色连衣裙 x Chanel 配饰搭配方案

- Notion page id: `2637a51c-1360-82fa-bd40-8193fa59b268`
- URL: https://www.notion.so/Skims-x-Chanel-2637a51c136082fabd408193fa59b268

用户已有一件 Skims 风格黑色修身长裙（方领、长袖、贴身线条）。

搭配目标：用 Chanel CF、Chanel 凉鞋和 Chanel 腰链打造高阶精致感。

图片款式分析：
- 版型：贴身直筒偏微喇裤脚线，视觉上拉长纵向比例。
- 领口：方领露锁骨，适合叠加细链/耳饰。
- 色彩：全黑极简，最适合做“配饰主导型”穿搭。

搭配方案（建议）：
1) 包：Chanel CF（中号/小号均可）
   - 晚间：黑金CF更稳重高级
   - 白天：奶白CF更轻盈
2) 鞋：Chanel凉鞋
   - 若裙摆贴地，优先有一点跟高，避免压身高
   - 若平底，建议露脚背款提升轻盈感
3) 腰链：Chanel chain belt
   - 建议放在自然腰略高1-2cm，强化胸腰比例
   - 若当天状态不想强调腰腹，可改挂包链+长耳饰替代

风格关键词：old money glam / minimal sexy / 黑金法式。

体态与穿着建议（友好版）：
- 这类裙不是“必须完美身材”才可穿，关键在于内搭与比例调整。
- 可用无痕塑形内搭、胸贴/无痕内衣、轻薄披肩提升安全感。
- 一套成片重点是：腰线位置+鞋跟高度+配饰金属统一。
